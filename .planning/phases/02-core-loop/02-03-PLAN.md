    **NOTE: All npm packages (zustand, @dnd-kit/*) are already installed by 02-01. Do NOT run npm install.**

    **Zustand store (`projects/briefs/command-centre/src/store/task-store.ts`):**
    - State: tasks (Task[]), isLoading (boolean), error (string | null)
    - Actions:
      - fetchTasks(): GET /api/tasks, populate store
      - createTask(title: string, level: TaskLevel): POST /api/tasks, add to store
      - updateTask(id: string, updates: TaskUpdateInput): PATCH /api/tasks/:id, update in store
      - moveTask(id: string, newStatus: string, newOrder: number): PATCH /api/tasks/:id with status and columnOrder, update in store optimistically (revert on error)
      - deleteTask(id: string): DELETE /api/tasks/:id, remove from store
      - applySSEEvent(event: TaskEvent): update store from SSE event (task:created adds, task:updated/task:status/task:progress patches, task:deleted removes)
    - Computed selectors:
      - getTasksByStatus(status): filter and sort by columnOrder
      - getChildTasks(parentId): filter tasks where task.parentId === parentId, sort by columnOrder
      - getRunningCount(): count tasks with status 'running'

    **SSE hook (`projects/briefs/command-centre/src/hooks/use-sse.ts`):**
    - Custom hook useSSE() that:
      - Creates EventSource connection to /api/events on mount
      - Parses incoming events and calls store.applySSEEvent()
      - Handles reconnection: on error, retry after 3 seconds with exponential backoff (max 30s)
      - Cleans up EventSource on unmount
      - Returns { isConnected: boolean }
