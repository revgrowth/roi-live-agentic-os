I just set up something in Claude Code that completely changed how I use it day to day. So Claude Code now has this slash loop feature, which basically lets you tell Claude to keep running a prompt on a timer. Every five minutes, every ten minutes, whatever you want. And the reason this is so useful is because I was spending a stupid amount of time just checking things manually. Is the build done? Did the tests pass? Has anyone replied to my PR? I'd just keep going back to the terminal, running the same command, waiting, going back. Now I just type slash loop, five m, check the build status, and Claude handles it. It runs in the background while I'm actually doing real work. And the crazy thing is it also works with slash commands, so I can loop any of my custom skills on a schedule. If you want to see exactly how I set this up, link's in my bio.
